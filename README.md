# Entregável IV

Você foi admitido para uma empresa e o cenário é o pior de todos: o sistema não está documentado, todos os devs foram demitidos (alguns até não estão mais entre nós!) e não tem ninguém para explicar como as coisas funcionam. No entanto, você precisa pagar as contas... e a tarefa que passaram-te foi a seguinte:

Reimplemente o sistema do zero! O motivo disto é que o sistema foi implementado em Visual J++ e está no ar sabe-se lá como! Até as chaves de acesso ao servidor foram perdidas, e o sistema está sendo pago via cartão de crédito de um estagiário que se mudou para a Indonésia e pode sair do ar a qualquer momento.

A única coisa que você sabe é que se você forkar e clonar este projeto do GitHub e rodar `npm install` (só uma vez) e depois `npm run atende-clientes`, é possível interagir com o sistema. Os requisitos para esta implementação são:

- utilizar um algoritmo de busca
- um algoritmo de ordenação
- uma estrutura de pilha
- uma estrutura de fila

Após finalizar, você deve colocar o código no GitHub e entregar o link via Classroom. Um dev desconhecido, que trabalhou na empresa antes de você (está desaparecido) criou um arquivo chamado `implemente-aqui.js`, que pode ser um bom ponto de partida!
